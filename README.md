Инструкция по установке "Системы прогнозирования заболеваний"

Вот подробная пошаговая инструкция по запуску проекта "Система прогнозирования заболеваний" (MedStat System) с нуля на операционной системе Windows.

Шаг 1: Установка Python

1. Так как у вас еще не установлен Python, начнем с него.

2. Перейдите на официальный сайт Python: https://www.python.org/downloads/

3. Нажмите большую желтую кнопку "Download Python 3.x.x" (скачается последняя версия).

4. Запустите скачанный установочный файл (.exe).

ВАЖНО: В появившемся окне установки обязательно поставьте галочку "Add python.exe to PATH" (Добавить Python в PATH) внизу окна. Без этого команда python не будет работать в командной строке.

5. Нажмите "Install Now" и дождитесь окончания установки.

Шаг 3: Настройка окружения

Теперь нам нужно создать изолированное место для библиотек (виртуальное окружение), чтобы не засорять систему.

Нажмите клавиши Win + R на клавиатуре.

В появившемся окне введите cmd и нажмите Enter. Откроется черное окно командной строки.

В командной строке нужно перейти в папку с проектом. Введите команду (замените путь на свой, если он отличается):

cd Desktop\DiseaseSystem


Создайте виртуальное окружение командой:

python -m venv venv


(После этого в папке проекта появится папка venv).

Активируйте виртуальное окружение командой:

venv\Scripts\activate


(Если все прошло успешно, в начале строки появится надпись (venv)).

Шаг 4: Установка библиотек

Программе нужны специальные модули (PyQt6 для интерфейса, pyqtgraph для графиков и т.д.). Они перечислены в файле requirements.txt.

Находясь в той же командной строке (с надписью (venv)), введите:

pip install -r requirements.txt


Дождитесь, пока побегут строки загрузки и установка завершится.

Шаг 5: Запуск программы

Все готово к запуску.

В командной строке введите:

python app.py


При первом запуске программа автоматически подключится к базе данных data/app_database.db. Если файла базы нет, он будет создан на основе schema.sql.

Введите эти данные и нажмите "Увійти" (Login).

Возможные ошибки

"python не является внутренней или внешней командой...": Вы забыли поставить галочку "Add Python to PATH" при установке. Переустановите Python, выбрав пункт "Modify" или удалив и установив заново с галочкой.

ModuleNotFoundError: No module named 'PyQt6': Убедитесь, что вы активировали виртуальное окружение (venv\Scripts\activate) перед тем как запускать python app.py.
